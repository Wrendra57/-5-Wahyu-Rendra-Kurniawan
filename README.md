## Endpoint

#### Get all cars

```http
  GET /api/cars
```

#### Get car by id

```http
  GET /api/cars/:id
```

#### Create car

```http
  POST /api/cars
```

#### Update cars

```http
  PUT /api/cars/:id
```
#### Delete cars

```http
  DELETE /api/cars
```
